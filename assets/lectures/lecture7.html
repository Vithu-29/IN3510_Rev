1. Frequency Reuse

Definition: Frequency reuse is the practice of assigning the same frequency bands to multiple cells in a cellular network, as long as the cells are placed far enough apart to avoid harmful interference.

Key Concepts

  • Channel Groups: Each cell is assigned a group of radio channels.
  • No Adjacent Reuse: Neighboring cells must use different channel groups to prevent interference.
  • Footprint: The coverage area of each cell, defined by a boundary where signals remain reliable.
  • Reuse in Distant Cells: The same channels can be reused in cells that are sufficiently far apart.

Why Frequency Reuse?

  • Increases Capacity: Enables more users to be served without needing new spectrum.
  • Efficient Spectrum Use: Makes better use of limited radio frequency resources.
  • Lower Cost: Minimizes need for additional spectrum licenses.
  • Essential for Large-Scale Networks: Supports nationwide coverage efficiently.

Example: 7-cell reuse pattern. Cells marked with the same letter reuse the same frequency set.

Frequency Reuse in 5G

  • 5G uses very high frequencies such as millimeter wave (mmWave).
  • These bands have short range and higher susceptibility to interference.
  • Frequent and tighter reuse is needed to maintain coverage and capacity.
  • Small cells make reuse more effective in dense areas.

2. Cluster Geometry

Cluster Size Formula (N)

For hexagonal cells, cluster size N is determined by non-negative integers i and j:

N = i2 + ij + j2

Possible values: 1, 3, 4, 7, 9, 12, 13, 19, 21...

N = Cells per cluster

1 / N = Frequency reuse factor

3. Interference

1. Co-Channel Interference

Interference from cells that reuse the same frequency band (co-channel cells).

  • Cannot be removed by increasing transmit power (also increases interference).
  • Depends on the reuse distance (D) between co-channel cells.
  • Smaller cluster size (N) → smaller D → stronger interference.
  • Measured using the co-channel reuse ratio: Q = D / R = √(3N).
  • Larger Q = lower interference.

2. Adjacent Channel Interference

Interference caused by signals on neighboring frequency channels.

  • Occurs due to imperfect analog/digital filters (frequency leakage).
  • Can come from the same base station or nearby mobile users.
  • Reduced by using guard bands between channels.
  • Also reduced by careful frequency allocation to avoid adjacent channels in the same cell.

Problem with Smaller Cluster Size

Smaller cluster size (N) increases capacity but also increases interference.

  • Co-channel cells become closer → co-channel interference increases.
  • Reuse distance (D) decreases → lower signal quality.
  • Coverage becomes less reliable, especially at cell edges.

Frequency Reuse Example

Problem:

Find the minimum distance (D) between the centers of two cells using the same frequency band, given:
Cell Radius (R) = 1 km
Frequency Reuse Factor (N) = 12

Formula: D / R = √(3N)

Rearranging: D = √(3N) * R

Substitution: D = √(3 * 12) * 1

Calculation: D = √36 * 1

Answer: D = 6 km

4. 3G Networks

Overview

3G (Third Generation) wireless systems were introduced in the early 2000s with the global standard defined by the ITU-T under IMT-2000 (International Mobile Telecommunications-2000). 3G significantly improved data speeds (144 kbps – 2 Mbps), enabling smartphones, multimedia applications, and mobile internet. Most regions adopted UMTS / W-CDMA, while others used CDMA2000.

GSM Evolution to 3G

  • GSM (CSD) – 9.6 kbps (circuit-switched, one time slot).
  • HSCSD – Uses up to 4 timeslots → ~50 kbps (good for real-time but inefficient).
  • GPRS – Packet-switched, up to ~115 kbps (no resource lock-up).
  • EDGE – Uses 8PSK → 3× faster than GPRS, up to ~384 kbps.
  • EGPRS – GPRS + EDGE combined for improved performance.
  • UMTS / W-CDMA (3G) reused GSM/GPRS core network for smooth migration.

CDMA vs W-CDMA

CDMA is a 2G/3G digital spread-spectrum technology, while W-CDMA (UMTS) is a true 3G system with wider bandwidth and higher capacity.

Features of 3G Technology

  • Faster communication (144 kbps – 2 Mbps).
  • Supports high-speed web browsing and secure connections.
  • Video calling, video conferencing, video streaming.
  • Supports 3D gaming and multimedia downloads.
  • Large capacity and broadband capabilities.
  • Soft handover → smoother movement between cells.
  • Better Quality of Service (QoS) for data and multimedia.

Drawbacks of 3G

  • Very high license fees for spectrum and infrastructure rollout.
  • Expensive 3G phones and larger device size in early stages.
  • High bandwidth requirement and complex technology.
  • Slow deployment and limited coverage during early years.
  • Battery consumption was significantly higher than 2G.

Issues in 3G Wireless Systems

  • High initial investment for 3G service licenses.
  • Multiple 3G standards (W-CDMA, CDMA2000, TD-SCDMA) caused fragmentation.
  • Inefficient for rapidly increasing mobile data demands.
  • Requires wider spectrum (5 MHz per carrier).
  • Lack of coverage during early deployment phase.
  • Competition from emerging 4G (LTE) made 3G lifespan shorter.

5. UMTS

Overview

UMTS (Universal Mobile Telecommunications System) is the 3G family standardized by 3GPP under IMT-2000. It is an umbrella term for mobile access technologies such as W-CDMA (FDD), TD-SCDMA and W-CDMA in TDD modes. UMTS supports both circuit-switched (CS) and packet-switched (PS) services and enables multimedia, mobile internet and higher data rates.

UMTS Domains & Key Components

  • User Equipment (UE): Mobile device and USIM. Has identities such as IMSI, MSISDN, IMEI.
  • UTRAN (UMTS Terrestrial Radio Access Network): Node B (base station) + RNC (Radio Network Controller). Handles radio access, mobility and radio resource management.
  • Core Network (CN): Split into Circuit-Switched (MSC/VLR/GMSC) and Packet-Switched (SGSN/GGSN) subsystems.
  • Support Databases: HLR (subscriber data), AuC (auth keys), EIR (device check).

Radio / Spectrum / Data Rates

  • Air interface: Wideband CDMA (W-CDMA) commonly uses 5 MHz carriers (chip rate 3.84 Mcps).
  • Typical UMTS data rates: rural ≈ 144 kbps | urban outdoor ≈ 384 kbps | indoor/low-range ≈ 2048 kbps
  • Frequency bands: Paired FDD bands (e.g. uplink/downlink pairs around 1900/2100 MHz; other regions use different pairs).
  • Duplex modes: FDD (frequency-duplex W-CDMA) and TDD variants (time-duplex).
  • Enhancements: HSPA (HSDPA/HSUPA) improves downlink/uplink throughput and latency (commonly referred to as 3.5G).

UTRAN — Node B & RNC Responsibilities

  • Node B (Base station): RF transmit/receive, physical layer processing, modulation/demodulation, power control on the air interface.
  • RNC (Radio Network Controller): Radio Resource Management (RRM), admission control, handover control (including soft handover), channel allocation, power control settings, ciphering and segmentation/reassembly of packets.
  • RNS (Radio Network Subsystem): One or more Node Bs controlled by an RNC; RNC–RNC interface is Iur.

Core Network — Packet & Circuit Functions

  • SGSN (Serving GPRS Support Node): Mobility & session management in PS domain. Creates PDP contexts, enforces QoS for data sessions, interacts with MSC/VLR and generates billing records (CDRs).
  • GGSN (Gateway GPRS Support Node): Gateway/router to external IP networks (Internet, corporate networks). Maintains mappings of IP addresses to subscriber PDP contexts and forwards IP packets to the correct SGSN.
  • MSC / VLR / GMSC: Circuit-switched switching, mobility for voice calls, and gateway to PSTN/PLMN.
  • Subscriber DBs: HLR (home subscriber info, services), AuC (authentication keys), EIR (blacklist/white list devices).

Important Interfaces, Identities & QoS

  • Key interfaces: Uu (UE ↔ UTRAN), Iub (RNC ↔ Node B), Iur (RNC ↔ RNC), Iu-PS/Iu-CS (UTRAN ↔ PS/CS core).
  • Subscriber & device identities: IMSI, TMSI, MSISDN, IMEI, IMEISV, P-TMSI, TLLI (used in different domains).
  • QoS & Bearers: Radio Access Bearers (RABs) & PDP contexts define guaranteed/assured QoS for services. QoS classes include conversational, streaming, interactive and background.

Security, Mobility & Virtual Home Environment

  • Security: Authentication (AuC/HLR), integrity protection and ciphering on radio links, and device checking via EIR.
  • Mobility: Fast handover techniques (soft handover in W-CDMA), inter-RNC handover, and roaming across PLMNs.
  • Virtual Home Environment (VHE): Allows subscriber service profiles and preferences to follow the user across different networks and terminals (personalized service environment).

Enhancements, Migration

HSPA (HSDPA / HSUPA): Incremental upgrades to UMTS that substantially improve data throughput and reduce latency. HSPA often marketed as 3.5G.

Backward compatibility: UMTS reuses GSM core network elements making migration easier for operators.
