3. FDMA Capacity & AMPS

FDMA Overview

  • Each transmitter is allocated a dedicated frequency channel with a specific bandwidth.
  • All transmitters can operate simultaneously and continuously.
  • FDMA can be used with both analog and digital signals, usually in narrowband systems.
  • Requires high-performance RF filters and duplexers to separate transmit and receive frequencies.
  • Continuous transmission requires fewer overhead bits (synchronization, framing) compared to TDMA.
  • System complexity is lower than TDMA but needs proper RF filtering to minimize adjacent channel interference.
  • Example: AMPS (Advanced Mobile Phone System) uses FDMA with 30 kHz channel bandwidth.

FDMA Capacity Formula

The number of channels (N) in an FDMA system is given by:

N = (Bt - 2Bguard)/Bch
  • Bt : Total spectrum allocation.
  • Bguard : Guard band (Unused part of the radio spectrum between radio bands, for the purpose of preventing interference).
  • Bch : Bandwidth of a single channel.

Example Calculation: AMPS System

Given:

  • System: FDMA/FDD (Analog)
  • Total Bandwidth ( Bt ) : 12.5 MHz
  • Guard Band ( Bguard ) : 10 kHz
  • Channel BW ( Bch ): 30 kHz

N = (12.5 x 10^6 - 2 x 10 x 10^3) / (30 x 10^3)

N = (12,500,000 - 20,000) / 30,000

N = 12,480,000 / 30,000

N = 416 Channels

4. TDMA Capacity & GSM

TDMA Capacity Formula

TDMA increases capacity by m times (number of time slots):

N = m(Btot - 2Bguard)/Bch
  • N : Number of TDMA channel slots.
  • m : Max number of TDMA users supported on each Bch
  • Btot : Total spectrum allocation.
  • Bguard : Guard band
  • Bch : Bandwidth of a single channel.

Ex 1: GSM Users

Given: Btot=25 MHz, Bch=200 kHz, m=8 slots. (Assume no guard band).

N = (8 * 25,000,000) / 200,000

N = 200,000,000 / 200,000

N = 1000 Simultaneous Users

Ex 2: GSM Timing

Given: Data Rate = 270.833 kbps, Bits/Slot = 156.25, Slots/Frame = 8.

  • (a) Bit Duration (Tb):
    1 / 270.833 kbps = 3.692 μs
  • (b) Slot Duration (Tslot):
    156.25 * 3.692 μs = 0.577 ms
  • (c) Frame Duration (Tf):
    8 * 0.577 ms = 4.615 ms
  • (d) Wait Time:
    Same as Frame Duration = 4.615 ms

5. Multiplexing Dimensions

Multiplexing allows multiple users to share a communication medium efficiently with minimal interference. It can happen in 4 dimensions: Space (Si), Time (t), Frequency (f), and Code (c).

Spread Spectrum: A method where the transmitted signal frequency is deliberately varied, resulting in a much wider bandwidth than the original signal.

Advantages:

  • Reduces the effect of catastrophic interference from other signals on the same frequency.
  • Makes signals harder to intercept, improving confidentiality.

Conventional Wireless Problems:

  • Constant-frequency signals are easily interfered with.
  • Constant-frequency signals are easy to intercept.

Multiplexing Goal: Efficiently share a medium among multiple users while minimizing interference. Guard spaces or guard bands are often required to prevent overlap between users.
